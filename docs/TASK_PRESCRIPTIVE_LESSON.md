## 0. Суть запроса пользователя

Сделать урок **строго предписывающим**: ребёнок не выбирает текст сам, а делает
всё **чётко по инструкции**. Главные требования:

1. **Точные указания «найди — замени».** В задании показываем *полную строку
   кода* и говорим заменить метку на *конкретное значение*. Пример из запроса:
   > Найдите `<div class="logo">🛍️ [ВПИШИ НАЗВАНИЕ МАГАЗИНА]</div>` и замените
   > `[ВПИШИ НАЗВАНИЕ МАГАЗИНА]` на текст **МегаМагазин**.
   Никакого «например, СуперМаркет / ИгроМир» — только одно точное значение.
2. **Заполнение значений в CSS.** В шаблоне CSS тоже есть метки-пропуски, куда
   ребёнок по инструкции вписывает **точный цвет**, **размер шрифта** и иногда
   другие свойства (border-radius, фон). Метка вида `[ВПИШИ ЦВЕТ]`.
3. **Ввод тегов через Emmet.** В **каждом HTML-шаге** есть подзадача: щёлкнуть на
   пустую строку, **впечатать готовую Emmet-аббревиатуру и нажать Tab** — добавить
   новый тег/элемент. Так ребёнок вписывает в HTML не только текст, но и теги.
4. **Аудитория:** дети 12–16 лет, **впервые видят код**, времени **1–1.5 часа**.
   Инструкции — максимально подробные и пошаговые.

### Известные нюансы (для реализации)

- **Метка в CSS делает свойство временно невалидным** (`color: [ВПИШИ ЦВЕТ];`).
  Это нормально и задумано: пока не вписан цвет — браузер игнорирует свойство,
  как впишет — увидит результат после ▶ Запустить. Жёлтая подсветка метки помогает
  её найти.
- **Цветовой чип** в CSS-редакторе появится сам, когда ребёнок впишет `#rrggbb`
  (3–6 hex-цифр) — это бонус, ломать не нужно.
- **Строки-литералы:** все поля шага — JS-строки. Короткие поля — в двойных
  кавычках (`"…"`, переносы `\n`), многострочные сниппеты — в шаблонных литералах
  (бэктики). Внутри сниппетов НЕ должно быть символов `` ` `` и `${` (см. PITFALLS).
- **Emmet-аббревиатуры не должны содержать `$`** (в `emmet.js` `$` — нумерация).
  В наших аббревиатурах `$` нет (цена пишется как `2990 ₽`, не `$`).
- **Пустая строка под Emmet-комментарием** должна иметь нужный отступ (2 пробела)
  и сохраниться в шаблонном литерале — НЕ обрезать trailing-пробелы у этой строки.

## 4. Как встроить Emmet-подшаг в сниппет

В HTML-сниппет каждого шага 1–6 добавляем **строку-комментарий + пустую строку с
отступом** в том месте, где должен появиться новый элемент (внутри нужного
контейнера, перед его закрывающим тегом). Шаблон (отступ = отступ соседних
элементов контейнера):

```html
  <!-- 👇 EMMET: щёлкни на пустую строку ниже, впиши код из задания и нажми Tab -->
  
```

Вторая строка — **пустая, но с 2 пробелами отступа** (чтобы Emmet раскрылся с
правильным отступом). В `taskMd` подробно объясняем: открой index.html → найди
строку с этим комментарием → щёлкни мышкой на пустую строку под ним → впиши/скопируй
аббревиатуру → нажми **Tab**.

Куда класть комментарий в каждом шаге:
- Шаг 1: внутри `<header>`, перед `</header>` (promo появится в шапке).
- Шаг 2: внутри `<section class="banner">`, перед `</section>`.
- Шаг 3: внутри `<section class="products">`, после 4-й карточки, перед `</section>`.
- Шаг 4: внутри `<section class="about">`, перед `</section>`.
- Шаг 5: внутри `<div class="review-list">`, после 3-го отзыва, перед `</div>`.
- Шаг 6: внутри `<footer>`, перед `</footer>`.

```markdown
## На каждом шаге делай 5 действий по порядку

- 1️⃣ **Вставь блок** кнопками: **HTML** уходит в index.html, **CSS** — в style.css.
- 2️⃣ Открой код и найди жёлтые метки **[ВПИШИ …]** — и в HTML, и в CSS.
- 3️⃣ **Замени каждую метку точно так, как написано в задании** (стирай метку
  целиком, вместе со скобками **[** и **]**).
- 4️⃣ **Добавь тег через Emmet:** найди строку с комментарием `<!-- 👇 EMMET … -->`,
  щёлкни на пустую строку под ним, **впиши код из задания и нажми Tab**.
- 5️⃣ Нажми **▶ Запустить** — и увидишь, как магазин растёт!

💡 Если после запуска на сайте видна надпись **[ВПИШИ …]** — значит, там ты ещё не
вписал текст. Ошибся? Нажми **Ctrl+Z** — действие отменится. Поехали! 👇
```

### Шаг 1 — Шапка

- **goalMd:** `**Цель:** добавить верхнюю часть сайта — *шапку*.\n\n**Новый термин:** \`<header>\` — это HTML-тег для «шапки» страницы, где живут логотип и меню.`
- **actionMd:** `Нажми обе кнопки: **HTML** добавит разметку шапки, **CSS** — её стили.`
- **taskMd:**
```markdown
🔧 **Сделай по порядку.**

**1. Название магазина (index.html).** Найди строку:
`<div class="logo">🛍️ [ВПИШИ НАЗВАНИЕ МАГАЗИНА]</div>`
и замени `[ВПИШИ НАЗВАНИЕ МАГАЗИНА]` на текст: **МегаМагазин**

**2. Цвет шапки (style.css).** Найди строку `background: [ВПИШИ ЦВЕТ];`
и замени `[ВПИШИ ЦВЕТ]` на: **#5b3df5**

**3. Размер логотипа (style.css).** Найди строку `.logo { font-size: [ВПИШИ РАЗМЕР]; ... }`
и замени `[ВПИШИ РАЗМЕР]` на: **24px**

**4. Добавь тег через Emmet (index.html).** Найди строку `<!-- 👇 EMMET … -->`,
щёлкни на пустую строку под ним, впиши этот код и нажми **Tab**:
`p.promo{🔥 Бесплатная доставка от 1000 рублей}`
```
- **hintMd:** `Метку стирай полностью, вместе со скобками \`[\` и \`]\`. Цвет вписывай ровно как в задании: \`#5b3df5\`. Размер — \`24px\` (число и буквы px без пробела).`
- **doneMd:** `✅ **Готово, когда** после ▶ Запустить вверху виден логотип «🛍️ МегаМагазин», меню, «🛒 Корзина: 0» и строчка про бесплатную доставку (нигде нет [ВПИШИ …]).`
